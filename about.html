About PCB Variation

PCB Variation is my project about assembly bounty + temporal physical fingerprinting. I want to prove a board is real by how it’s built and how it drifts over time. It’s not a portfolio, it’s one focused experiment that I’m building in public.

Back to the project Talk to me

What assembly bounty means

If a board is actually built and tested, it should earn proof. The bounty part means the hardware itself helps verify the assembly so the work can be trusted.

Temporal physical fingerprinting

I measure tiny timing and analog shifts that happen as a board warms up, cools down, and ages. That time-based drift is the fingerprint I’m chasing.

Why I’m doing this

Counterfeits and fake builds are a real thing. I want a cheap, repeatable way to prove a board is legit using physics instead of just a sticker.

How I’m building it

  1. Plan the fingerprints. Pick signals that drift in a measurable way.
  2. Build the boards. Same design, different real-world quirks.
  3. Capture the data. Log time-series traces, not just one snapshot.
  4. Analyze the drift. Find the patterns that stay unique.
  5. Prove the build. Tie it all together in a bounty-style proof.
