what is this thing?

this is a project im working on (i like to call it assembly bound temporal physical fingerprinting) where i test what happens when the exact same microcontroller runs the same code on different pcbs.

im trying to see if the pcb itself actually matters in tiny timing ways.

why im doing this

i couldnt find anyone online doing this exact thing so i decided to just do it myself. also i like hardware and overthinking stuff.

why this is nerdy

if this works, the pcb becomes part of the identity of the device. not the software. not the chip. the physical board.
